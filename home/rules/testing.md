# テストルール

- テストコードは `tests/` 以下に、`src/` と同じ構成で配置する
- 主要機能のユニットテストを作成し、エッジケースを考慮する
- モックは最小限に留め、実際の機能テストを優先する
- 実行は `uv run pytest`（カバレッジは `uv run pytest --cov=src`）

## 完了条件

- テストを作成・変更したら必ず `uv run pytest` を実行する
- **すべてのテストが成功するまで修正を続ける**。1つでも失敗があれば作業完了とみなさない
